# LCM Finder in Python

This is a simple Python program that calculates the **Least Common Multiple (LCM)** of two numbers entered by the user.

## 📌 What is LCM?

The **Least Common Multiple (LCM)** of two integers is the smallest positive number that is divisible by both numbers.

Example:
LCM of 4 and 6 = 12

lcm.py - Python script to calculate LCM
